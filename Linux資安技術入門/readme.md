# 課程宗旨

# 課程內容
- A.基礎linux入門
  - 認識Linux作業系統:多人使用與多工作業
  - 作業系統功能
  - linux 核心(kernels)與Shell
  - Linux 發行版本(distributions)
  - Linux的目錄結構
- B.Linux基本指令
  - Linux-檔案與目錄操作指令
  - redirection(重定向)與pipe(管道)
  - Linux網路指令
- C.隱寫術入門
  - 認識 隱寫術 Steganography 
  - 文件隱寫術 : WORD隱寫術 PDF隱寫術
  - 圖片隱寫術 :圖片中的隱藏資訊
    - 檔案格式(file signature)
    - 使用hex編輯器分析檔案
    - 使用Linux的dd、binwalk指令解圖片隱寫術
- D.網路封包分析入門
